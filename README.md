# async_blog

async_blog is a blog API service built with Go, this project for learning message processing like RabbitMQ, NATS and Kafka and PostgreSQL for data storage

# Run api
`
cd api
go run main.go
`
# Run consumer
`
cd consumer
go run main.go
`
